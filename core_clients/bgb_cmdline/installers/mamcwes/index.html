This page is all about...... mamcwes

What's that?

Fine working php class for mysql connectivity, with back and forward compatbility support concerning API calls, for php 4, php 5, mysql, mysqli client librarys.

For who is it?

PHP writers / developers / coders generally.

What does it do?

Besides the basic functionality one would expect from a mysql(i) class one could send a userlevel variable to it, for example from a logged in user, to select a specific mysql connection.
Also the content from the config file will only be loaded to the php memory when nessecary and immediately unloaded afterwards.

How does it work?

$this->gl_conf_ulvl in the mysqlcfgdata.inc.php is the userlevel setting. Default it loads from $_SESSION['rdmstring'].
One should use the escstr function to parse any kind of user input.
Looking at the class (functions) name(s), it won't be too hard figuring out how to make the DB connection / calls.
It supports php4 with mysql client library
It supports php5 with both mysql and mysqli client library
